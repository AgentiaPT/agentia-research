Signal Radar

  • 🔴
    Anthropic: two leaks in one week + Congressional scrutiny
    Mythos model specs via CMS misconfiguration, Claude Code source via npm packaging error. Rep. Gottheimer questions national security implications.
  • 🔴
    Supply chain attacks weaponize security tools
    TeamPCP compromised Trivy vulnerability scanner, KICS, LiteLLM. The detection-defense loop breaks when scanners become attack vectors. 1,000+ environments affected.
  • 🔴
    Axios npm compromise: 70M+ weekly downloads poisoned
    Credential theft + RAT installation via poisoned versions 1.14.1 and 0.30.4. Dual version strategy targeted both modern and legacy codebases.
  • 🟠
    AI agents choose vulnerable dependencies 50% more often
    Study of 117K dependency changes: AI coding agents systematically favor popularity over security when selecting package versions.
  • 🟠
    Oracle 6am email: mass layoff template for AI era
    30K employees fired simultaneously via email at dawn. No managers, no HR. $8-10B freed for AI data center buildout.
  • 🟠
    DMCA overreach: 8,100 repos nuked in 24 hours
    Anthropic's takedown hit legitimate forks of their own public repo. Retracted, but demonstrates copyright enforcement risk at scale.
  • 🟢
    Gemma 4 under Apache 2.0 — Google's open model play
    31B model ranks #3 globally. Apache 2.0 eliminates legal friction. On-device AI rivaling cloud models from 2025.
  • 🟢
    Copilot cloud agent: research → plan → code
    GitHub's agent now understands codebases, produces implementation plans, executes with signed commits and org governance.
  • 🟢
    OpenClaw: agent-as-universal-interface pattern
    Karpathy's Dobby dynamically discovers and controls arbitrary networked systems. One agent replacing 6+ specialized apps.
  • 🔵
    Claude Mythos "Capybara" tier — step-change capabilities
    Leaked specs describe unprecedented cyber capabilities. Anthropic reportedly briefing US officials on risks.
  • 🔵
    "AI-washing" layoffs narrative splits mainstream
    Andreessen, Fortune, and Bloomberg publish competing analyses. Is AI replacing workers, excusing layoffs, or funding infrastructure?

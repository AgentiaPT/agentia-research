## Story: OpenAI Ends Microsoft Azure Exclusivity, Models Land on AWS Bedrock
- **Date:** 2026-04-28
- **Source:** https://www.cnbc.com/2026/04/28/openai-brings-models-to-aws-after-ending-exclusivity-with-microsoft.html
- **Outlet:** CNBC
- **Key facts:**
  - OpenAI officially ended its ~7-year exclusive hosting agreement with Microsoft Azure
  - GPT-5.5, Codex, and frontier models now available on Amazon Bedrock in limited preview
  - Backed by a reported $38–50 billion multi-year cloud commitment between OpenAI and AWS
  - Microsoft retains "primary" cloud partner status via non-exclusive license valid through 2032; Azure still gets early access to new releases

- **Why it matters:** Ends the biggest cloud lock-in in AI, giving enterprises multi-cloud choice for OpenAI workloads and intensifying the AWS-Azure-GCP competition.
- **Confidence:** high
